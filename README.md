# Restaurant App

A responsive restaurant application built with Flutter.

## Features
- **Menu View:** Browse delicious menu items with descriptions and prices.
- **Shopping Cart:** Add items to your cart, adjust quantities, or remove them entirely.
- **Checkout Flow:** Simple checkout dialog and cart clearing mechanism.
- **Responsive Design:** Adaptive grid layout for mobile, tablet, and desktop views.

## Running the App
Make sure you have Flutter installed, then run:
```bash
flutter run
```
